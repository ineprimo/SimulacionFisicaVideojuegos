CONTROLES:
Movimiento del jugador: WASD
Movimiento de los objetos (aspersor y estiércol): IJKL, en el mismo orden que WASD
Cambiar el objeto para mover:
	1 -> para mover el aspersor
	2 -> para mover el estiércol
Activar/desactivar la bomba: B
Activar/desactivar el estiércol: P
Activar/desactivar el aspersor: Y
Aumentar la masa del sistema actual: O
Disminuir la masa del sistema actual: U
